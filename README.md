# 🐙 BenoQs Sequencer

> *A reimagining of the iconic GenoQs Octopus (2008), rebuilt in Max/MSP and Ableton Live.*

BenoQs is not just a sequencer. It's an instrument for exploration.

---

## ✨ Overview

The BenoQs Sequencer draws its inspiration from the philosophy and design of the legendary GenoQs Octopus: a performance-driven, deeply interactive sequencing environment that encourages experimentation at every level.

This project recreates that spirit in a modern hybrid setup using Max/MSP and Ableton, with a focus on:

- Non-linear sequencing
- Unconventional timing and tempo structures
- Hands-on, tactile exploration
- Deep integration with modular hardware

---

## 🧠 Philosophy

At its core, BenoQs follows one simple idea:

> *No matter what stage you're at, you are encouraged to experiment, explore, and push boundaries.*

This isn't about precision sequencing alone. It's about discovering patterns, embracing unpredictability, and building systems that surprise even their creator. The sequencer is as much a compositional collaborator as it is a tool.

---

## ⚙️ Features

| Feature | Description |
|---|---|
| 🌀 Multi-layered sequencing | Grid, Page, Track, and Step hierarchy with per-level parameter inheritance |
| ⏱️ Experimental timing | Non-standard tempo multipliers, groove/shuffle, step push/pull offsets |
| 🎛️ Modular-first design | Built for interaction with hardware modular gear via MIDI |
| 🔁 Pattern evolution | Non-traditional step behaviour including skip, brownian motion, and ping-pong directions |
| 🎵 Polyphonic step chords | Up to 7 notes per step with independent polyphony and random note pool picks |
| 🧩 Max/MSP + Ableton | Native Max for Live device with full transport and clock sync |

## 🔌 Use Cases

- Experimental electronic music

---

## 🛠️ Requirements

- Ableton Live 11+ (Suite or with Max for Live)
- Max for Live 8+
- A MIDI interface or virtual MIDI bus (e.g. IAC Driver on macOS) for hardware routing

---

## 🚀 Getting Started

1. Clone or download this repository
2. Drag `BenoQs.amxd` into any MIDI track in Ableton Live
3. Set your MIDI output in the device to route to your instrument or hardware
4. Press play and start exploring

Full documentation is in the [Wiki](../../wiki) *(coming soon)*.
